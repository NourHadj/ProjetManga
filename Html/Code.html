Notre code

La fonction Navigate return le code htmml de l'url entrée, ainsi qu'une liste contenant toutes les images du chapitre.

La fonction RecupListeLienS prend argument
le soup de l'url(le code html ) et nous return
tout les liens des scans du chapitre.

Démonstration

La fonction Next prend en argument l'url et nous return
l'url suivant. Mais si l'url entrée est la dernière url alors
la fonction nous return "Fin du Manga".

La fonction soop prend en argument une url et nous return son soup.
